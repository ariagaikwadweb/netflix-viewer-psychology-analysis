# 🎬 Netflix Viewer Psychology Analysis

A data storytelling and audience psychology analysis project exploring Netflix content trends, binge behaviour, genres, and emotional themes using Python and data visualization.

---

# 🎯 Project Objective

This project aims to explore:

- Netflix content distribution
- Viewer binge psychology
- Genre dominance and emotional storytelling
- Global streaming trends
- Dark vs non-dark content patterns

The analysis combines data analytics with behavioural interpretation to understand how streaming platforms shape viewer engagement.

---

# 🛠 Tools Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook
- GitHub

---

# 📊 Key Insights

## Movies Dominate Netflix
Netflix contains significantly more movies than TV shows, indicating a movie-heavy content strategy.

## International Content Growth
Countries like India and South Korea contribute heavily to Netflix’s global catalogue expansion.

## Dark Emotional Themes Are Common
A significant portion of Netflix content contains emotionally intense themes such as crime, violence, revenge, and war.
